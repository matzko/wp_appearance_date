== Description ==

Show a post on a date separate from its publish date.

== Installation ==

1. Upload the `wp-appearance-date` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Set an appearance date for a post when editing that post.

== Frequently Asked Questions ==

= Why would someone want to do this? =

If you’re a content publisher, such as a magazine, you may want to show a post on a date different from when it’s associated. For example, a post for a March issue might show up in February.
